	Hello! Welcome to the Kilburnazon Employee Management System!
	Please choose the function in the left navigation bar to start management.
	If you want to return to this weclome page, you could click the "Main Page".
	

	Function Description:

	If you want to add new employee information, click "Add new Employee".

	If you want to update the salary and emergency phone number, click "Update Information".

	If you want to delete the employee information, click "Delete an Employee".

	"Driver's ECP" will show drivers' information whose emergency relationship is "Father".

	If you want to know who have the birthday this month, just click "Birthday List".

	"Senior Delete" will ask you to input your ID when deleting an employee.

	For operations making changes to the database, you will get the message at the top of websites.

	For operations showing the data, you will see results at the bottom of websites.
